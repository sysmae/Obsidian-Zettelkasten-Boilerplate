# 참고 노트

그냥 복붙하는게 아니라 자신만의 언어로 정리

Reference note
 - 타인의 생각이나 저작물 기록, 인용문 강조문, 하이라이트해서 저장
### 언급한 노트 (Outgoing Links)

```dataview

LIST

FROM -"Templates"

WHERE contains(this.file.outlinks, file.link) AND file.name != this.file.name

```
